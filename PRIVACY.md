# Privacy Policy

AirChat — локальное (офлайн) приложение для обмена сообщениями поблизости. Приложение может использовать Bluetooth и локальную сеть для обнаружения устройств и передачи данных без интернета.

## Данные
- Сообщения и вложения, которые вы создаёте/отправляете.
- Технические метаданные доставки (идентификаторы сообщений, статусы доставки).

## Чего приложение не делает
- Не требует серверной регистрации по умолчанию.
- Не продаёт персональные данные.

## Разрешения iOS
- Bluetooth / Local Network: офлайн‑обмен сообщениями.
- Microphone / Photo Library / Camera: только для функций медиа по вашему действию.
